# 📱 KeenKeeper-Platform

## 🧾 Project Name
KeenKeeper-Platform

---

## 📝 Short Description

Friend Timeline is a React-based web application The homepage is the main entry point of the Friend Timeline app. It provides a clean dashboard that introduces the app and gives quick access to Timeline, Friend Details, and Stats. The design is simple, responsive, and  helps users track their interactions with friends such as Calls, Texts, and Video calls. Each interaction is stored in a dynamic timeline with filtering and analytics support.

---

## 🛠️ Technologies Used
- React.js
- Context API
- JavaScript (ES6+)
- Tailwind CSS
- React Icons
- React Toastify
- Recharts

---

## ⭐ Key Features

### 1. 📊 Interactive Timeline
Track all friend interactions (Call, Text, Video) in a real-time timeline sorted by latest activity.

---

### 2. 🔍 Smart Filtering System
Easily filter interactions by type:
- Call
- Text
- Video
- All

---

### 3. 📈 Analytics Dashboard
Visual representation of interactions using a Pie Chart:
- Shows Call vs Text vs Video distribution
- Dynamic updates based on user actions

---

## 🏠 Application Flow
1. User lands on Homepage
2. Navigates to Friend Details page
3. Clicks Call / Text / Video buttons
4. Interaction is added to Timeline
5. Stats update automatically
6. Timeline shows latest activity on top
